ML vs AI

What is machine learning?

(One) definition:

A computer program is said to learn from experience E with respect to some task T and some performance measure P, if its performance on T, as measured by P, improves with experience E.

Tom Mitchell, 1997

When should we use machine learning?

ML is good when we have access data, which is

  • descriptive of the task we want to solve
  • available in big¬π amounts

[1] (We will get back to how much this is)

ML is less useful when

  • the data are incomplete or inconsistent
  • the data have simple correlations
  • the underlying (data generating) process can be modelled analytically

ML is bad when

  • we need an explainable / interpretable model
  • under certain requirements of privacy
  • the data change (in an unforeseen way) over time
  • the developer don‚Äôt know what they are doing

Different types of machine learning

Supervised learning:

Each datapoint is assigned to a label, which the model tries to predict.

Label is a

  • class (categorical):
    we are doing classification
  • value (numerical):
    we are doing regression

Different types of machine learning

Supervised learning:

Each datapoint is assigned to a label, which the model tries to predict.

Unsupervised learning:

Datapoints are unlabelled, but the model tries to group similar ones, or otherwise learn a pattern.

Reinforcement learning:

The model (aka agent) interacts with an environment, and receives rewards or penalties depending on its actions
