Linear regression (again)

Letâ€™s set up some math for regression with a simple linear model, on data with \(n\) features:

\[ \small \begin{align} \hat{\color{DarkBlue}{y}} &= \color{Purple}{f}(\color{DarkOrange}{\mathbf{x}}, \boldsymbol{\color{teal}{\theta}}) \\ &= \color{teal}{\theta}_0 + \color{teal}{\theta}_1 \color{DarkOrange}{x}_1 + \cdots + \color{teal}{\theta}_n \color{DarkOrange}{x}_n \end{align} \]

Compactly written on vector form:

\[ \small \hat{\color{DarkBlue}{y}} = \boldsymbol{\color{teal}{\theta}}^{\intercal} \cdot \color{DarkOrange}{\mathbf{x}} \]

For one-dimensional data, it looks like

\[ \small \begin{align} \boldsymbol{\color{teal}{\theta}}^{\intercal} \cdot \color{DarkOrange}{\mathbf{x}} &= \begin{bmatrix} \color{teal}{\theta}_0 & \color{teal}{\theta}_1 \end{bmatrix} \begin{bmatrix} \color{DarkOrange}{x}_0 \\ \color{DarkOrange}{x}_1 \end{bmatrix} \\ &= \color{teal}{\theta}_0 \color{DarkOrange}{x}_0 + \color{teal}{\theta}_1 \color{DarkOrange}{x}_1 \qquad \color{grey}{(\mathrm{with}\; x_0=1}) \\ &= \color{teal}{\theta}_0 + \color{teal}{\theta}_1 \color{DarkOrange}{x}_1 \end{align} \]

Finding optimal parameters \(\boldsymbol{\theta}\)

Step 1: Define optimal through a loss function \(\color{MediumVioletRed}{L}\)

For regression, mean squared error is the most common

\[ \small \begin{align} \color{MediumVioletRed}{L}_{\mathrm{MSE}} &= \frac{1}{N}\sum_{i=1}^{N} \left(\hat{\color{DarkBlue}{y}}^{(i)} - \color{DarkBlue}{y}^{(i)}\right)^2 \\[1em] &= \frac{1}{N}\sum_{i=1}^{N} \left(\boldsymbol{\color{teal}{\theta}}^{\intercal} \color{DarkOrange}{\mathbf{x}}^{(i)} - \color{DarkBlue}{y}^{(i)}\right)^2 \end{align} \]

Need to differentiate the loss wrt. each parameter:

\[ \small \begin{align} \frac{\partial \color{MediumVioletRed}{L}_{\mathrm{MSE}}}{\partial\color{teal}{\theta}_j} &= \frac{\partial}{\partial\color{teal}{\theta}_j} \left(\frac{1}{N}\sum_{i=1}^N (\boldsymbol{\color{teal}{\theta}}^{\intercal}\color{DarkOrange}{\mathbf{x}}^{(i)} - \color{DarkBlue}{y}^{(i)})^2 \right) \\[1em] &= \frac{1}{N}\sum_{i=1}^N 2(\boldsymbol{\color{teal}{\theta}}^{\intercal}\color{DarkOrange}{\mathbf{x}}^{(i)} - \color{DarkBlue}{\mathbf{y}})\color{DarkOrange}{x}_j^{(i)} \end{align} \]

Gradient descent

Put it all together to get the gradient
(written here in compact form)

\[ \small \nabla \color{MediumVioletRed}{L}(\boldsymbol{\color{teal}{\theta}}) = \frac{2}{N}\color{DarkOrange}{\mathbf{X}}^{\intercal}(\color{DarkOrange}{\mathbf{X}} \cdot \boldsymbol{\color{teal}{\theta}} - \color{DarkBlue}{\mathbf{y}}) \]

where \(\color{DarkOrange}{\mathbf{X}}\) contains all the training data:

\[ \small \color{DarkOrange}{\mathbf{X}} = \begin{bmatrix} \color{DarkOrange}{\mathbf{x}}^{(1)} \\ \color{DarkOrange}{\mathbf{x}}^{(2)} \\ \vdots \\\ \color{DarkOrange}{\mathbf{x}}^{(N)} \end{bmatrix} \]

Gradient descent

With the gradient in place, we take steps downward (along the negative gradient), towards the optimal solution:

\[ \small \boldsymbol{\color{teal}{\theta}}^{n+1} = \boldsymbol{\color{teal}{\theta}}^n - \eta \nabla \color{Purple}{L} \]

Here \(\eta\) is the learning rate

In code, this looks like

theta = np.random.randn(2,1)
for epoch in range(n_epochs):
  gradients = 2/N * X.T.dot(X.dot(theta) - y)
  theta = theta - eta*gradients

Gradient descent methods

Standard (batch) gradient descent:

Compute \(\small\nabla \color{MediumVioletRed}{L}\) using all data points

\[ \small \boldsymbol{\theta}^{n+1} = \boldsymbol{\theta}^n - \eta\nabla L (\boldsymbol{x}, \boldsymbol{\theta}) \]

Gradient descent methods

Stochastic gradient descent:

Compute \(\small\nabla L\) using a single data point at a time

\[ \small \boldsymbol{\theta}^{n+1} = \boldsymbol{\theta}^n - \eta\nabla L (x_i, \boldsymbol{\theta}) \]

Gradient descent methods

Mini-batch gradient descent:

Compute \(\small\nabla L\) using some data points at a time

\[ \small \boldsymbol{\theta}^{n+1} = \boldsymbol{\theta}^n - \eta\nabla L (\boldsymbol{x}_s, \boldsymbol{\theta}) \]

Automatic differentiation

Manually computing gradients is rather inconvenient ðŸ˜’

Luckily, automatic differentiation helps us out:

import jax

def predict(X, theta):              # For complicated models, predict()
  return X.dot(theta)               # also tends to get very complicated

def loss(X, theta, y):              # But loss function is always simple
  y_hat = predict(X, theta)
  mse = ((y_hat - y)**2).mean()
  return mse

for epoch in range(n_epochs):
  theta = theta - eta*jax.grad(loss, argnums=1)(X, theta, y)  # autodiff!
