Finding optimal parameters

Step 1: Define optimal

Need a metric to measure how well the function fits the data

In regression tasks, mean squared error is typically used:

\[ \small \mathrm{MSE}(\mathbf{y},\hat{\mathbf{y}}) = \frac{1}{N} \sum_{i=1}^{N} (y_i - \hat{y}_i)^2 \]

Step 2: Use an optimisation algorithm to find the best parameters

  • For linear regression, there is a closed-form solution
  • For most other methods we need a numerical approach
    (more on this later)

Hyperparameters

In the past example we choose a 3rd degree polynomial.

  • Why not degree 2 or 4 or something else?

From the monomial basis we can build any model

\[ \small \hat{y} = \color{Purple}{f}(\color{DarkOrange}{x}, \boldsymbol{\color{teal}{\theta}}) = \color{teal}{\theta}_0 + \color{teal}{\theta}_1 \color{DarkOrange}{x} + \color{teal}{\theta}_2 \color{DarkOrange}{x}^2 + \dots + \color{teal}{\theta}_M \color{DarkOrange}{x}^M \] where \(M \in \mathbb{N}\) is a hyperparameter:

Chosen by us, and not the optimisation algorithm.

Regularisation

  • Our main goal: Minimise the difference between prediction \(\hat{y}\) and true value \(y\), i.e. the \(\mathrm{norm}(\hat{y} - y)\)
  • Side goal: Minimise the parameter values, i.e. minimise \(\mathrm{norm}(\mathbf{\theta})\)

Several ways to compute the norm:

\(\small{}L0\) norm:

Number of nonzero parameters

\[ \small L0([2,3]) = 2 \]

\(\small{}L1\) norm:

Sum of parameter absolute values

\[ \small \begin{align} L1([2,3]) &= 2 + 3 \\ &= 5 \end{align} \]

L1 regularisation aka
Lasso regression

\(\small{}L2\) norm:

Euclidian distance

\[ \small \begin{align} L2([2,3]) &= \sqrt{2^2 + 3^2} \\ &= \sqrt{13} \end{align} \]

L2 regularisation aka
Ridge regression aka
Tikhonov regularisation aka
weight decay

Regularisation (cont.)

Piece together loss and regularisation term, and minimise simultaneously:

\[ \small \mathrm{loss}(\hat{y}, y) + \color{red}{\alpha} \cdot \mathrm{norm}_p (\mathbf{\theta}) \]

where the regularisation parameter \(\color{red}{\alpha}\) controls the strength of regularisation

No free lunch (NFL) theorem

There is (provably) no single machine learning method that will work best on all problems.

In other words:

Under no assumptions about the data, there is no reason to choose a certain ML method (or any optimisation technique in general) over another.
