Linear regression

Let‚Äôs set up some math for regression with a simple linear model, on data with \(n\) features:

\[ \small \begin{align} \hat{\color{DarkBlue}{y}} &= \color{Purple}{f}(\color{DarkOrange}{\mathbf{x}}, \boldsymbol{\color{teal}{\theta}}) \\ &= \color{teal}{\theta}_0 + \color{teal}{\theta}_1 \color{DarkOrange}{x}_1 + \cdots + \color{teal}{\theta}_n \color{DarkOrange}{x}_n \end{align} \]

Compactly written on vector form:

\[ \small \hat{\color{DarkBlue}{y}} = \boldsymbol{\color{teal}{\theta}}^{\intercal} \cdot \color{DarkOrange}{\mathbf{x}} \]

For one-dimensional data, it looks like

\[ \small \begin{align} \boldsymbol{\color{teal}{\theta}}^{\intercal} \cdot \color{DarkOrange}{\mathbf{x}} &= \begin{bmatrix} \color{teal}{\theta}_0 & \color{teal}{\theta}_1 \end{bmatrix} \begin{bmatrix} \color{DarkOrange}{x}_0 \\ \color{DarkOrange}{x}_1 \end{bmatrix} \\ &= \color{teal}{\theta}_0 \color{DarkOrange}{x}_0 + \color{teal}{\theta}_1 \color{DarkOrange}{x}_1 \qquad \color{grey}{(\mathrm{with}\; x_0=1}) \\ &= \color{teal}{\theta}_0 + \color{teal}{\theta}_1 \color{DarkOrange}{x}_1 \end{align} \]

Nonlinear regression bonus slide

Humble linear regression:

\[ \small \hat{\color{DarkBlue}{y}} = \boldsymbol{\color{teal}{\theta}}^{\intercal} \cdot \color{DarkOrange}{\mathbf{x}} \]

Can make it non-linear by applying a so-called activation function \(h\):

\[ \small z = h(\boldsymbol{\color{teal}{\theta}}^{\intercal} \cdot \color{DarkOrange}{\mathbf{x}}) \]

for example

\[ \small z = \tanh(\boldsymbol{\color{teal}{\theta}}^{\intercal} \cdot \color{DarkOrange}{\mathbf{x}}) \]

Regularised linear models

Let‚Äôs improve our linear model by adding a regularisation term:

\[ \small \color{MediumVioletRed}{L}(\color{teal}{\boldsymbol{\theta}}) = \color{MediumVioletRed}{L}_{\mathrm{MSE}} + \color{red}{\alpha} ||\color{teal}{\boldsymbol{\theta}}||_p \]

  • \(\small L_2\) regularisation, aka Ridge regression(sklearn.linear_model.Ridge):

\[ \small \color{MediumVioletRed}{L}(\color{teal}{\boldsymbol{\theta}}) = \color{MediumVioletRed}{L}_{\mathrm{MSE}} + \color{red}{\alpha} ||\color{teal}{\boldsymbol{\theta}}||_2 \]

\[ \small \nabla \color{MediumVioletRed}{L}(\color{teal}{\boldsymbol{\theta}}) = \nabla \color{MediumVioletRed}{L}_{\mathrm{MSE}} + \color{red}{\alpha} \color{teal}{\boldsymbol{\theta}} \]

  • \(\small L_1\) regularisation, aka Lasso regression (sklearn.linear_model.Lasso):

\[ \small \color{MediumVioletRed}{L}(\color{teal}{\boldsymbol{\theta}}) = \color{MediumVioletRed}{L}_{\mathrm{MSE}} + \color{red}{\alpha} ||\color{teal}{\boldsymbol{\theta}}||_1 \]

(\(\small\nabla\color{MediumVioletRed}{L}\) gets a little complicated ‚Äì see Eq. 4.11 in the textbook)

Hyperparameter optimisation

Technique for finding optimal hyperparameters:

Make intelligent guesses
(usually cannot be formulated as a differentiable task)

  • Grid search:
    Test a fixed set of combinations
  • Random search:
    Test randomly selected combinations
  • Bayesian optimisation / evolutionary algorithms / etc.:
    Cleverly select new combinations based on past results
